# ごちノート（飲食店料理記録アプリ）

## 概要
ごちノート は、飲食店で食べた「料理」を記録・管理するための Web アプリケーションです。  
料理名、店舗名、評価、メモ、画像を登録することで、食事の記録を簡単に振り返ることができます。

本アプリは卒業制作として、フロントエンドからバックエンドまで一貫して個人で開発しました。

---

## 使用技術
- Next.js（App Router）
- TypeScript
- Supabase（Database / Storage / Server Actions）
- CSS Modules
- Vercel（デプロイ）　[リンク] ( https://vercel.com/ryuukumatas-projects/restaurant-record2 )

## 操作説明

### 1. ホーム画面
アプリを起動すると、登録済みの料理リスト一覧が表示されます。  
料理リストは登録日時の新しい順に並びます。
一番上のタイトルとサブタイトルの下に検索窓があり、料理名やタグ、感想の一部を入れることで該当する料理リストだけが表示されます。
検索窓の下には今まで登録したデータのタグが表示され、タグをクリックすると押したタグに該当している料理のリストだけが表示されます。
表示されている料理のリストをクリックするとクリックした料理の詳細画面に飛びます。
下の新規登録ボタンをクリックすると新規登録画面にとびます。

---

### 2. 料理の登録
ホーム画面の新規登録ボタンをクリックすると、料理登録画面に遷移します。 
左の矢印をクリックするとホーム画面に戻ります。 
以下の情報を入力して登録できます。

- 料理名
- 店舗名
- 評価（★）
- メモ
- 料理画像

登録した情報は Supabase に保存され、一覧画面に反映されます。

---

### 3. 詳細画面
一覧から料理のリストをクリックすると、詳細画面が表示されます。
左の矢印をクリックするとホーム画面に戻ります。  
登録した画像やメモを確認することができます。
最後の部分に編集ボタンと削除ボタンがあり、編集ボタンをクリックすると編集画面に削除ボタンをクリックすると保存した内容が削除されてホーム画面に遷移します。

---

### 4. 編集画面
詳細画面の編集ボタンをクリックすると、編集画面が表示されます。
左の矢印をクリックすると詳細画面に戻ります。
料理登録の情報を編集して変更することができる。
更新するというボタンをクリックすると更新しましたって表示が上に現れ、変更された表示画面が出てきます。

---

## 工夫した点

- 料理画像を Supabase Storage に保存し、URL で管理している点
- シンプルで直感的な操作を意識した UI 設計
